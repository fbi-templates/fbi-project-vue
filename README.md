# fbi-project-vue
A modern project which uses `Vuejs` and `Webpack`.

## Requirements

```bash
fbi v3.0+
node v7.6+
```

## Usage
```bash
$ fbi init https://github.com/fbi-templates/fbi-project-vue.git new-project  
           
$ fbi s                      
```
